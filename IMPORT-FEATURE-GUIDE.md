# 地区管理导入功能使用指南

## 功能概述

地区管理模块现在支持完整的Excel文件导入功能，用户可以通过下载标准模板、填写数据、上传文件的方式批量导入地区信息。该功能与下载的模板完全保持一致，确保数据格式的统一性和准确性。

## 功能特性

### 1. 标准模板下载
- **模板格式**：Excel 2007+ (.xlsx) 格式
- **模板内容**：包含完整的字段说明、示例数据和数据验证
- **字段定义**：6个标准字段，3个必填，3个可选

### 2. 智能文件解析
- **格式支持**：支持 .xlsx 和 .xls 格式
- **数据验证**：自动验证必填字段和数据类型
- **错误处理**：友好的错误提示和异常处理
- **空行跳过**：自动跳过空行和无效数据

### 3. 批量数据导入
- **事务处理**：支持事务回滚，确保数据一致性
- **重复检查**：自动检查地区编码重复性
- **结果反馈**：详细的导入结果统计和错误信息

## 使用流程

### 第一步：下载模板
1. 进入地区管理页面
2. 点击"下载模板"按钮
3. 浏览器自动下载"地区导入模板.xlsx"文件

### 第二步：填写数据
1. 打开下载的Excel文件
2. 参考示例数据填写地区信息
3. 注意必填字段的标识（*）
4. 使用下拉选择填写级别和状态

### 第三步：上传导入
1. 点击"导入数据"按钮
2. 将填写好的Excel文件拖拽到上传区域
3. 或点击上传区域选择文件
4. 点击"开始导入"按钮
5. 等待导入完成并查看结果

## 模板字段说明

| 列名 | 字段说明 | 是否必填 | 数据类型 | 验证规则 | 示例值 |
|------|----------|----------|----------|----------|--------|
| 地区编码* | 地区唯一标识 | 是 | 文本 | 1-20位数字，唯一 | 110000 |
| 地区名称* | 地区显示名称 | 是 | 文本 | 2-50字符 | 北京市 |
| 地区级别* | 地区行政级别 | 是 | 数字 | 1-3，下拉选择 | 1 |
| 父级编码 | 上级地区编码 | 否 | 文本 | 6位数字或空 | 110000 |
| 排序 | 显示顺序 | 否 | 数字 | 整数，默认0 | 1 |
| 状态 | 启用状态 | 否 | 数字 | 0或1，下拉选择 | 1 |

## 数据填写规则

### 1. 必填字段
- **地区编码**：必须为6位数字，且在整个系统中唯一
- **地区名称**：2-50个字符，建议使用标准地名
- **地区级别**：1-省，2-市，3-区

### 2. 可选字段
- **父级编码**：上级地区的6位编码，省级地区可不填
- **排序**：数字越小显示越靠前，默认值为0
- **状态**：0-禁用，1-启用，默认值为1

### 3. 数据关系
- 省级地区：级别=1，父级编码为空
- 市级地区：级别=2，父级编码为省级编码
- 区级地区：级别=3，父级编码为市级编码

## 常见问题解决

### 1. 文件格式问题
**问题**：上传非Excel文件
**解决**：只支持 .xlsx 和 .xls 格式，请使用Excel软件保存

### 2. 数据格式问题
**问题**：地区编码不是6位数字
**解决**：确保地区编码为6位数字，如 110000

**问题**：地区级别不在1-3范围内
**解决**：使用下拉选择，只能选择1、2、3

**问题**：状态值不是0或1
**解决**：使用下拉选择，只能选择0或1

### 3. 数据重复问题
**问题**：地区编码已存在
**解决**：检查是否重复填写，或修改为新的编码

### 4. 父级关系问题
**问题**：父级编码不存在
**解决**：确保父级地区已存在，或先导入父级地区
