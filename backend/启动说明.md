# 权限管理系统 - IDEA启动说明 (Java 8版本)

## 🚀 快速启动

### 1. 环境要求
- **Java版本**: Java 8 (JDK 1.8)
- **Maven版本**: Maven 3.6+
- **数据库**: MySQL 5.7+

### 2. 数据库准备
- 确保MySQL服务已启动
- 执行 `src/main/resources/db/init.sql` 创建数据库和表

### 3. IDEA启动步骤

#### 方法一：直接运行主类
1. 在IDEA中打开项目
2. 找到 `src/main/java/com/permission/PermissionApplication.java`
3. 右键选择 "Run 'PermissionApplication.main()'"
4. 等待启动完成

#### 方法二：Maven插件启动
1. 在IDEA右侧找到Maven面板
2. 展开 `Lifecycle` → `spring-boot:run`
3. 双击执行

### 4. 启动成功标志
```
=================================
权限管理系统启动成功！
后端服务地址: http://localhost:8080
API文档地址: http://localhost:8080/swagger-ui.html

```

### 5. 测试接口
- 健康检查: http://localhost:8080/api/test/health
- 测试接口: http://localhost:8080/api/test/hello
- Swagger UI: http://localhost:8080/swagger-ui.html

## ⚠️ 注意事项

1. **Java版本**: 确保使用Java 8版本 (JDK 1.8)
2. **Maven版本**: 建议使用Maven 3.6+
3. **数据库**: 确保MySQL连接正常，用户名密码正确
4. **端口占用**: 确保8080端口未被占用
5. **Spring Boot版本**: 使用2.7.18版本，兼容Java 8

## 🔧 常见问题

### 问题1: 数据库连接失败
- 检查MySQL服务是否启动
- 检查 `application.yml` 中的数据库配置
- 确保数据库和表已创建

### 问题2: 端口被占用
- 修改 `application.yml` 中的 `server.port` 配置
- 或者关闭占用端口的其他服务

### 问题3: 依赖下载失败
- 检查网络连接
- 尝试使用国内Maven镜像源
- 清理Maven缓存: `mvn clean install`

### 问题4: Java版本不匹配
- 确保IDEA项目SDK设置为Java 8
- 检查JAVA_HOME环境变量指向JDK 1.8
- 确保Maven使用Java 8编译
